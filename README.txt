
#randam bot#

discordチャット内であらかじめした文字列に対して、

反応し内部的に抽選を行います。

（内部抽選の確率はソースコード編集で行ってください。）



数字がtrue場合に特定の文字列を返すbotです。

elseだった場合固定された文章を返します。
