
#randam bot#


randam要素を取り入れたかっただけですが、

discordチャット内であらかじめした文字列に対して、

反応し内部的に抽選を行います。

（内部抽選の確率はソースコード編集で行ってください。）



数字がtrue場合に特定の文字列を返すbotです。

elseだった場合固定された文章を返します。

----------------------------------------

初めに
discord.py install手順等

https://github.com/Rapptz/discord.py/blob/async/README.md





ソースのrandam部分は1〜20の数字を選ぶようになっているので

デフォルトでは20分の1の確率設定になっています。

trueでの返す文字列とelseで返す文字列はソースコードに書き加えてください

例　true "当たり"
　　else "全然だめだな"
